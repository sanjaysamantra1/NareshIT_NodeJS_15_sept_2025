Function Currying
=================
-transforming a function that takes multiple arguments into a series of function that take 1 arg at a time.
    function add(a,b,c){ // normal function
        console.log(a+b+c)
    }
    function add(a){ // function currying
        return function(b){
            return function(c){
                console.log(a+b+c)
            }
        }
    }
