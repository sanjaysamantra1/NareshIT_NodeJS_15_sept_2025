How to execute Javascript Files
===============================
1. Browser
    -Create a Javascript file , add javascript code/content (addition.js)
    -Include that javascript file in an HTML file (addition.html)
     <script src='addition.js'></script>
    -Open HTML File in Browser
      a. add 'open in browser' extension in vscode
      b. alt + b   (OR)  alt + shift + b

2. NodeJS
    -Create a Javascript file , add javascript code/content (addition.js)
    -Run that javascript file in nodejs, run the below command
      node <filename>.js (make sure nodejs is installed)

Note : Every Javascript file that runs on browser may not work in nodeJS
       Every Javascript file that runs on Nodejs may not work in browser

Objects Available both in browser & nodejs : console , Math, Array , String
Objects Available only in browser,not in nodejs : window , document 
Objects Available only in nodejs,not in browser : os, fs, http, url 



            Browser                             NodeJS
====================================================================         
1. DOM(document) - yes                  1. DOM(document) - No
2. window - yes                         2. window - No
3. cookie - yes                         3. cookie - No
4. offline storage - yes                4. offline storage - No
5. FileSystem - No                      5. FileSystem - yes
6. Database Interaction - No            6. Database Interaction - yes
7. Client App / FrontEnd/ UI            7. Server App / BackEnd
8. Consumes APIs                        8. Develops APIs


NodeJS : Develop APIs , fetch the live data from database & returns that tdata to client/FrontEnd
FrontEnd : Consumes the APIs, collects the data from NodeJS and displays on the screen.
